# Technology Stack

## Frontend (Client)

| Technology | Purpose |
| :--- | :--- |
| **React** | Component-based UI library. |
| **Vite** | Next-generation frontend tooling and bundler. |
| **Tailwind CSS** | Utility-first CSS framework for rapid styling. |
| **Shadcn/UI** | Accessible, reusable component primitives (Radix UI based). |
| **Axios** | Promise-based HTTP client for API requests. |
| **React Router** | Declarative routing for SPAs. |

## Backend (Server)

| Technology | Purpose |
| :--- | :--- |
| **Node.js** | JavaScript runtime environment. |
| **Express** | Fast, unopinionated web framework for Node.js. |
| **MongoDB** | NoSQL database for flexible data schemas. |
| **Mongoose** | ODM (Object Data Modeling) library for MongoDB. |
| **Passport.js** | Authentication middleware (Google OAuth Strategy). |
| **Multer** | Middleware for handling `multipart/form-data` (audio uploads). |

## AI & External Services

| Service | Function |
| :--- | :--- |
| **Google Gemini 1.5 Flash** | **Brain:** Sentiment analysis, natural language generation, entity extraction. |
| **Groq (Whisper)** | **Ears:** Ultra-fast speech-to-text transcription. |
| **OpenAI Edge TTS** | **Voice:** High-quality, localized text-to-speech synthesis (e.g., `en-IN-NeerjaNeural`). |
| **OpenWeatherMap** | **Context:** Real-time weather conditions and forecasts. |
| **TomTom Traffic** | **Context:** Real-time traffic flow and congestion data. |
