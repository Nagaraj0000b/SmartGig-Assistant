# Welcome to Project Documentation

**Project** is a next-generation AI companion designed specifically for India's gig economy workers (Uber, Swiggy, Rapido, etc.). It acts as a smart digital assistant that helps workers track earnings, log shifts, and receive personalized operational advice.

## Key Features

-   **🗣️ Voice-First Interface:** Interact with the chatbot naturally using Hinglish (Hindi + English) voice commands.
-   **🌦️ Context-Aware Intelligence:** Real-time integration with Weather and Traffic APIs for smarter suggestions.
-   **💰 Earnings Analytics:** Consolidated view of income across multiple platforms.
-   **🧠 Burnout Detection:** Mathematically analyzes daily mood to protect workers from severe mental fatigue.
-   **🚀 Modern Tech Stack:** Powered by React, Node.js, MongoDB, and state-of-the-art AI models (Gemini + Groq).
